# GitHub Actions Demo

A simple npm package demonstrating basic TypeScript setup with testing and linting.

## Installation

```bash
npm install github-actions-demo
```


## Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the package:
   ```bash
   npm run build
   ```
4. Run tests:
   ```bash
   npm test
   ```
5. Run linting:
   ```bash
   npm run lint
   ```
